ooknight ui
==============================================

# qucik start (maven)

### jquery 2
```xml
<dependency>
    <groupId>com.github.ooknight.ui</groupId>
    <artifactId>jquery2</artifactId>
    <version>2.2.4.0</version>
</dependency>
```
```html
<script src="/static/plugins/jquery/jquery-2.2.4.min.js"></script>
```
